Detailed Description

Barcamp Transparency Object.

This class represents the root class for all objects in the framework and provides a helpful wrapper for data manipulation.

Most every object that you use will be a subclass of this object (or more accurately subclass of one of its subclasses) (Object, Relationship, Annotation etc.

Each class has a hierachy, set by a call to setType() in its constructor. This, combined with a call to the parent constructor defines the type of the object and a unique identifier for this class for the database queries.
