getObjectByUrl ( $  url  ) 

Get an object by a URL.

Attempt to retrieve an object by a URL, you can provide a way to match custom objects to URL by using the hook object/getbyurl, otherwise the function assumes the default export url.

Rationale

All objects in the BCT system have a getURL() method to return an address by which the object can be accessed (and by specifying a view=xxxx parameter on this url can be accessed in a variety of different formats). This function allows this process to go full circle and take a previously exported address of an object and return the object it references.

Parameters:
string $url
Returns:
BCTObject|false

string|array $type The type, based on the hierachy, use '' as a wildcard and ":" as delimiter. Eg. Use 'news' to return only news objects.
