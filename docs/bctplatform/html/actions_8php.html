Detailed Description

Actions.

Actions are endpoints for web forms and provide ways of modifying data rather than passing variables to pages in the traditional way.

The advantages of this are numerous, but among them are:

1) You call functions not pages 2) The framework provides protection against a number of scripting attacks.
